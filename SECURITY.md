# Security Policy

##  Supported Versions

The following versions of the project are currently supported with security updates:

| Version        | Supported |
| -------------- | --------- |
| Latest         |  Yes     |
| Older versions |  No      |

---

##  Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

###  How to report

* Open an issue (if the vulnerability is not sensitive), or
* Contact the maintainer directly via email (if applicable)

###  What to include

Please provide as much information as possible:

* Description of the vulnerability
* Steps to reproduce
* Possible impact
* Suggested fix (if available)

---

##  Security Considerations

This project includes:

* Camera input (computer vision)
* Microphone input (speech recognition)

Please be aware:

* Ensure proper permissions are handled securely
* Avoid exposing sensitive data
* Validate all external inputs

---

##  Disclaimer

This project is provided for educational and research purposes.
While basic security practices are followed, it may not be fully hardened for production environments.

---

##  Responsible Disclosure

We appreciate responsible disclosure of vulnerabilities and will address them as soon as possible.
